# Pangloss-Scraper
A Python script to download the recording files (.wav/.mp4 and .xml) of a given language, currently only from http://lacito.vjf.cnrs.fr/pangloss/index_en.html.

Requires the installation of [BeautifulSoup](https://www.crummy.com/software/BeautifulSoup/bs4/doc/#installing-beautiful-soup) and [Requests.](https://pypi.org/project/requests/)
